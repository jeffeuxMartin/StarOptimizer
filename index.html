Instructions

For f(t), type the number after ee. Use the price of the next cheapest f(t) purchase. Make sure you have purchased all variables and upgrades that cost less than this input.

For stars, make sure this is the number of stars you can spend only on permanent levels, i.e., excluding variable and automation purchases.

For Supremacy upgrade, enter the highest level of the third Supremacy upgrade (the one that initially costs 1e110ψ). You won't see this upgrade unless you bought all levels of the "y exponent" upgrade. Leave the field blank if you don't have access to this upgrade yet or if you want it to be deduced from f(t).
